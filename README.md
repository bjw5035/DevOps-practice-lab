## 소개
- 이 저장소는 DevOps 연습용 랩입니다. Docker와 Kubernetes 예제가 포함되어 있습니다.
- 주요 디렉터리 개요:
  - docker: docker-compose 파일과 관리 스크립트
  - kubernates: ConfigMap, Secret, Pod, Ingress 등 쿠버네티스 매니페스트
  - linux_shell: 간단한 리포트/모니터링 스크립트
